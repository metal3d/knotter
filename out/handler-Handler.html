Handler

new Handler()

Handler that will match route to method. This is the class to implement on project. Minimum required is "route" and one of HTTP method as get, post, put, delete... Handler has this properties to work: - params.args => that matches regexp on url - params.get => options given by ?argname=argvalue&... - postdata => object having post data - response => to write response - request => the current request
