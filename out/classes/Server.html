Server Class

Defined in: lib/httpserver.js:19
Module: knotter

Server is HTTP Server. To be used, prepare some handlers then call constructor with options as:

  • handlers: Array of handlers
  • statics: Array or Object to map some directory as static container
  • port: to listen
  • address: ...

Example:

 var s = new knotter.Server({
     handlers: [hdl1, hdl2],
     statics: {'styles': 'css/directory/'},
     address : '0.0.0.0',
     port    : 8080
 });
