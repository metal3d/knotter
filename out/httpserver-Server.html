Server

new Server(options)

Server is HTTP Server. To be used, prepare some handlers then call constructor with options as: - handlers: Array of handlers - statics: Array or Object to map some directory as static container - port: to listen - address: ...
Parameters:
Name Type Description
options Object (port, handlers, statics...)
